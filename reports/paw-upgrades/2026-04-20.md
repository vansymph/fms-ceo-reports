### Top Frameworks in 2026

**Zep** — Long-term memory store for conversational AI. Episodic + semantic memory separation with automatic consolidation.

**Letta (formerly MemGPT)** — Stateful agents with persistent OS-like memory architecture. Core concepts around stateful agents now well-documented.

**Cognee** — Builds evolving knowledge graphs per user. Tracks history, preferences, behavioral patterns across long-term conversations.

**File-first memory pattern** (validated by community):
- MEMORY.md = semantic (facts, preferences, long-term context)
- Daily logs = episodic (what happened, what was decided)
- Consolidation logic = deduplication across sessions
